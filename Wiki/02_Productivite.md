# ⭐ Astuce du Senior

## Terminal 

Je regarde systématiquement trois choses quand j'ouvre un terminal :

* où je suis (pwd)
* qui je suis (whoami)
* sur quelle machine je suis (hostname)

Cette habitude évite énormément d'erreurs.

---

## Environnement de travail

* Terminal à gauche
* Documentation à droite

Mon organisation personnelle quand je travaille :

* Écran gauche : VS Code
* Écran droit : 4 terminaux
* Navigateur : Documentation

Je touche très rarement à la souris.
