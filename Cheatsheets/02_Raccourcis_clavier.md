# 🧰 Boîte à outils du Senior : Raccourcis clavier

La touche super correspond à la touche windows

## Les indispensables :

### Environnement travail

**Ouvrir un terminal :**
```
Ctrl + Alt + T
```

**Rechercher une application :**
```
Super  puis taper son nom
```

**Changer d'application :**
```
Alt + Tab
```

**Verrouiller la session :**
```
Super + L
```

**Mettre une fenêtre à gauche :**
```
Super + <-
```

**Mettre la fenêtre à droite :**
```
Super + ->
```

**Mettre en plein écran :**
```
Super + ↑
```

---

### Bash

**Historique : **
```
↑
```

**Recherche intelligente :**
```
Ctrl + R
```

* Début de ligne : Ctrl + A
* Fin de ligne : Ctrl + E
* Effacer jusqu'au début : Ctrl + U
* Effacer jusqu'à la fin : Ctrl + K
* Effacer un mot : Ctrl + W
* Interrompre une commande : Ctrl + C
