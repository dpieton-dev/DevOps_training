🧪 TP 2 — Les 20 raccourcis à maîtriser

Je ne veux pas que tu les lises.

Je veux que tu les pratiques.

Exercice 1

Ouvre un terminal uniquement avec :

Ctrl + Alt + T
Exercice 2

Ferme-le avec :

Ctrl + D

Puis rouvre-le.

Exercice 3

Tape :

echo Bonjour Dominique

Puis :

Ctrl + A
Ctrl + E
Ctrl + U
Ctrl + K

Observe ce qui se passe.

Exercice 4

Tape plusieurs commandes :

ls
pwd
date
whoami
history

Puis utilise :

Ctrl + R

Recherche :

who

Tu retrouveras immédiatement :

whoami
Exercice 5

Teste l'autocomplétion :

cd Doc

Puis :

Tab
Exercice 6

Affiche deux fenêtres :

Terminal
Firefox

Passe de l'une à l'autre avec :

Alt + Tab
Exercice 7

Mets le terminal à gauche :

Super + ←

Puis Firefox à droite :

Super + →

Tu viens de créer une disposition de travail très utilisée.

🎯 Challenge du chapitre

Pendant 30 minutes, essaie de ne pas toucher la souris.

Utilise uniquement :

le clavier ;
les raccourcis ;
le terminal.

Au début, tu seras un peu plus lent.

Dans quelques jours, tu seras nettement plus rapide.
